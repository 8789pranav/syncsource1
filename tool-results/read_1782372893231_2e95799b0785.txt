    12‚Üí    12‚Üí## Architecture Decisions (Phase 1)
    13‚Üí    13‚Üí- **Stack**: Next.js 16 App Router, TypeScript, Tailwind v4, shadcn/ui (New York), Prisma (SQLite), Recharts, Zustand.
    14‚Üí    14‚Üí- **Single route**: only `/` is user-visible. The app is a client-side SPA-style shell with a sidebar; modules render as sections driven by a Zustand store (`activeModule`).
    15‚Üí    15‚Üí- **Theme**: emerald/teal primary on a slate-neutral base (NO indigo/blue). Light + dark mode via next-themes.
    16‚Üí    16‚Üí- **Dynamic form engine**: forms are defined as JSON schemas stored in DB (`FormSchema` model) and rendered by a generic `<DynamicForm>` React component. Every "Add X" modal uses this engine. A Form Builder lets admins create/edit schemas.
    17‚Üí    17‚Üí- **APIs**: all under `/api/*` using route handlers + Prisma. Single default tenant (`TENANT_ID` constant) for Phase 1.
    18‚Üí    18‚Üí- **Workflows**: `Workflow` + `WorkflowStep` models; `WorkflowInstance` tracks each request's progress. Leave/asset/etc. requests flow through workflows.

    97‚Üí    97‚Üí### API conventions (MANDATORY)
    98‚Üí    98‚Üí- Every model has `tenantId`. Always call `const tenantId = await ensureTenant()` first in handlers.
    99‚Üí    99‚Üí- LIST endpoints: `GET /api/<resource>` ‚Üí returns `{ items: [...] }`
   100‚Üí   100‚Üí- CREATE: `POST /api/<resource>` with JSON body ‚Üí returns created object
   101‚Üí   101‚Üí- UPDATE: `PATCH /api/<resource>/<id>` ‚Üí returns updated object
   102‚Üí   102‚Üí- DELETE: `DELETE /api/<resource>/<id>` ‚Üí returns `{ ok: true }`
   103‚Üí   103‚Üí- PICKER endpoints: `GET /api/<resource>/picker` ‚Üí returns `{ items: [{label, value}] }` (used by dynamic form pickers)
   104‚Üí   104‚Üí- Use `NextRequest`/`Request` from next. Import `db` from `@/lib/db`.
   105‚Üí   105‚Üí
   106‚Üí   106‚Üí### Dynamic form usage pattern
   107‚Üí   107‚Üí```tsx
   108‚Üí   108‚Üíimport { DynamicForm } from "@/components/dynamic-form/dynamic-form"
   109‚Üí   109‚Üíimport { getDefaultSchema } from "@/lib/form-schemas"
   110‚Üí   110‚Üíconst schema = getDefaultSchema("employee")!
   111‚Üí   111‚Üí<DynamicForm schema={schema} initialValues={row} onSubmit={async (v) => { await fetch('/api/employees', {method:'POST', body: JSON.stringify(v)}) }} onCancel={...} />
