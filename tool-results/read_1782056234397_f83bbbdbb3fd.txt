    12→    12→## Architecture Decisions (Phase 1)
    13→    13→- **Stack**: Next.js 16 App Router, TypeScript, Tailwind v4, shadcn/ui (New York), Prisma (SQLite), Recharts, Zustand.
    14→    14→- **Single route**: only `/` is user-visible. The app is a client-side SPA-style shell with a sidebar; modules render as sections driven by a Zustand store (`activeModule`).
    15→    15→- **Theme**: emerald/teal primary on a slate-neutral base (NO indigo/blue). Light + dark mode via next-themes.
    16→    16→- **Dynamic form engine**: forms are defined as JSON schemas stored in DB (`FormSchema` model) and rendered by a generic `<DynamicForm>` React component. Every "Add X" modal uses this engine. A Form Builder lets admins create/edit schemas.
    17→    17→- **APIs**: all under `/api/*` using route handlers + Prisma. Single default tenant (`TENANT_ID` constant) for Phase 1.
    18→    18→- **Workflows**: `Workflow` + `WorkflowStep` models; `WorkflowInstance` tracks each request's progress. Leave/asset/etc. requests flow through workflows.

    97→    97→### API conventions (MANDATORY)
    98→    98→- Every model has `tenantId`. Always call `const tenantId = await ensureTenant()` first in handlers.
    99→    99→- LIST endpoints: `GET /api/<resource>` → returns `{ items: [...] }`
   100→   100→- CREATE: `POST /api/<resource>` with JSON body → returns created object
   101→   101→- UPDATE: `PATCH /api/<resource>/<id>` → returns updated object
   102→   102→- DELETE: `DELETE /api/<resource>/<id>` → returns `{ ok: true }`
   103→   103→- PICKER endpoints: `GET /api/<resource>/picker` → returns `{ items: [{label, value}] }` (used by dynamic form pickers)
   104→   104→- Use `NextRequest`/`Request` from next. Import `db` from `@/lib/db`.
   105→   105→
   106→   106→### Dynamic form usage pattern
   107→   107→```tsx
   108→   108→import { DynamicForm } from "@/components/dynamic-form/dynamic-form"
   109→   109→import { getDefaultSchema } from "@/lib/form-schemas"
   110→   110→const schema = getDefaultSchema("employee")!
   111→   111→<DynamicForm schema={schema} initialValues={row} onSubmit={async (v) => { await fetch('/api/employees', {method:'POST', body: JSON.stringify(v)}) }} onCancel={...} />
   112→   112→```
